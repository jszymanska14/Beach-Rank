# Beach Rank

Aplikacja do porównywania i filtrowania plaż w Polsce.

## Funkcjonalności

- Interaktywna mapa z lokalizacjami plaż
- Sporty wodne (surfing, windsurfing, kitesurfing) z dedykowanymi ikonkami na mapie:
  - Surfing
  - Windsurfing
  - Kitesurfing
  - Wiele sportów dostępnych
- Zaawansowane filtrowanie po wielu parametrach:
  - Temperatura powietrza i wody
  - Siła wiatru
  - Obecność sinic
  - Jakość powietrza
  - Poziom zatłoczenia
  - Region
  - Sporty wodne (surfing, windsurfing, kitesurfing) - każdy osobno
- Porównywanie wybranych plaż
- Fullscreen dla mapy
- Responsywny design

## Instalacja

```bash
npm install
```

## Uruchomienie

```bash
npm run dev
```

Aplikacja będzie dostępna pod adresem `http://localhost:5173`

## Budowanie

```bash
npm run build
```

## Technologie

- React 18
- Vite
- Leaflet & React-Leaflet (mapy)
- CSS Modules
